About AfriDev Explorer

Shining a light on African developer talent

Our Mission

AfriDev Explorer is built to make African developer talent visible and accessible globally. The African tech ecosystem is vibrant, innovative, and growing — but African developers often struggle to gain international visibility and opportunities.

This platform aggregates profiles and open-source contributions from across the continent, making it easy to discover talent and celebrate African innovation.

Tech Stack

Built entirely with the fundamentals — no frameworks, no bloat — AfriDev Explorer demonstrates the power of vanilla web technologies:

  • HTML5 — Semantic markup and accessibility
  • Vanilla CSS — Custom design system with CSS variables, Grid, and Flexbox
  • JavaScript (ES6+) — Module-based architecture with modern features
  • GitHub REST API v3 — Public endpoints, free to use, rate-limited
  • Jest — Comprehensive unit testing framework
  • GitHub Actions — Automated testing and deployment pipeline
  • Progressive Enhancement — Works without JavaScript, enhanced with it
